# Changelog – Service Notification

## 1.0.0
- Version stable du service de notifications.
- Envoi automatique d’un email de bienvenue après inscription.
- Contenu de l’email : résumé des fonctionnalités de la plateforme.
- Intégration avec le service Auth via une requête API ou événement.

## 0.3.0
- Intégration d’un pipeline CI/CD (build, test, push Docker).
- Stockage des images sur GitHub Container Registry (GHCR).

## 0.2.0
- Ajout des tests unitaires pour le module d’envoi de mail.
- Création de la branche `tests`.
- Mock des services externes pour les tests.

## 0.1.0
- Initialisation du service.
- Configuration de l’envoi de mails via SMTP/API.
- Rédaction de l’email de bienvenue.
